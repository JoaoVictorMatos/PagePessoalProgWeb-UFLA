# Projeto: Página Pessoal - Programação Web (GAC116)

Este repositório contém a Atividade Prática 2 da disciplina de Programação Web da **Universidade Federal de Lavras (UFLA)**. O objetivo é desenvolver uma página pessoal responsiva, apresentando informações reais, projetos e suporte a Dark Mode em duas versões de estilização distintas.

## 📂 Estrutura do Projeto

O projeto foi dividido em dois diretórios principais para demonstrar a aplicação de diferentes frameworks na mesma base de conteúdo:

1.  **/versao-bootstrap**: Versão construída utilizando os componentes e sistema de grid do Bootstrap 5.
2.  **/versao-tailwind**: Versão construída utilizando a abordagem utility-first do Tailwind CSS.

Ambas as versões possuem:

- Layout totalmente responsivo.
- Seções de Sobre, Formação, Projetos e Contato.
- Alternador de tema (Light/Dark Mode) via JavaScript com persistência no `localStorage`.
